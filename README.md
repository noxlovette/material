# Ogonek M3

A Svelte 5 component library based on Material Design 3.

## Installation

```bash
bun add @ogonek/m3
```

## Usage

```svelte
<script>
  import { Button } from "@ogonek/m3";
  import "@ogonek/m3/styles";
</script>

<Button variant="filled">Hello World</Button>
```

## Development

```bash
bun install
bun run dev --open
```

## Building

```bash
bun run build
```
